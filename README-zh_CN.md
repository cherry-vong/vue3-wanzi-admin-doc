## 起源

2022 年 2 月 7 日，Vue 3 成为了新的默认版本。

市面上也出现了很多基于 Vue 3 开发的优秀的中后台解决方案，如 NaiveAdmin、vue-vben-admin、vue-admin-better 等，但是在这些解决方案相关的生态、跨端方案、安全方案等有所欠缺，于是我们便想打造一款拥有良好生态的企业级跨端前后台解决方案。

## 项目开发

版本要求：

- Node LTS 14+
- pnpm 6.x+/yarn/npm

安装依赖：

```sh
# 推荐
$ pnpm install

# 或者
$ yarn install

# 或者
$ npm install
```

运行项目：

```sh
$ pnpm run dev
```

打包：

```sh
$ pnpm run build
```

## 背景及说明

### 产品优势

- 开源版本免费商用
- 采用最新的技术栈 vue3+typescript+vite
- 完整的社区生态
- 基于 RBAC 及 JWT 鉴权机制
- 企业级前端安全解决方案
- 多主题、多种布局灵活切换
- 快速迭代
